Project Overview

My Netflix project is a recreation of the real web app home page. I've used Angular to build the structure and it was my first finished project with this technology. The database is self-done (a college gave me all the pictures). I've used node packages to add carrousels, and by giving them a big dose of CSS I've managed to get very close to the real look and feel of Netflix. It is responsive although I wasn't really practising this aspect in this project.

The aim of this project was to test Typescript and a real file structure in Angular.
