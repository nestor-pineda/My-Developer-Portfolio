Project Overview

"Sneakers" is the final project I did for the React sections of the Upgrade Hub Bootcamp. I developed this app store using the Sneakers API from RapidAPI. The idea was to recreate a store where you can add products to a cart and delete them if you finally decide not to buy them. You may add products to the cart but you need to be logged in to access the cart/dashboard section.

The aim of this project was to master how to fetch an external API with several parameters in the URL creating a reusable custom Hook. To play around with useContext and useReduce Hooks, to add some cool style effects using Styled Components and to develop a mobile-first responsive design.
