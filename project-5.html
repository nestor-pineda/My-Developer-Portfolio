Project Overview

With the Pokedex project, I learned how to deal with APIs and how to fetch information and create a loop to display it on my website. I tried different approaches and finally got used to the traditional fetch method that returns a promise. I added pagination using the offset and the number of items per page.

The aim of this project was to get experience with handling the information from the Pokemon API and returning results from an array of objects.
